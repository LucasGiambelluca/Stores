Nuevo Feature

Product Page Builder

Sistema de construcción visual para páginas de producto con widgets personalizables y acceso por niveles.

↓ Desliza para ver más

La Visión

Llevar la flexibilidad del Home Builder a la página más importante del e-commerce: el Producto.

  • Drag & Drop intuitivo
  • Widgets especializados (Reviews, Specs, Videos)
  • Diferenciación clara entre planes (Enterprise vs Pro)
  • Experiencia de usuario premium

Arquitectura del Sistema

1. Data Model

Nuevo esquema ProductPageConfig en la base de datos. Almacena el array de bloques y configuraciones globales (colores, espaciado).

2. Block Renderer

Adaptación del BlockRenderer actual para soportar contexto de producto (pasar product prop a los widgets).

3. Widget System

Descomposición de ProductDetail.tsx en micro-componentes: GalleryWidget, BuyBoxWidget, SpecsWidget.

Estrategia de Monetización
